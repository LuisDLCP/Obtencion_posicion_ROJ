What's about?
This script extracts position data from a NMEA file.
	INPUT FILES: *.201
	OUTPUT_FILES: *.pos  
-------
Instructions:

1) Place the NMEA files in the path: '/Obtencion_posicion/Input_files/Data_set/' 
2) Run the script 'nmea2position.py'
2) The output files will be stored in two paths: 
		'/home/cesar/Desktop/luisd/scripts/Obtencion_posicion/Output_files'
		'/home/cesar/Desktop/luisd/scripts/Obtencion_posicion/Output_files/ToUpload/'
